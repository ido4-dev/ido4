# Plugin Moved

The ido4dev Claude Code plugin is now in its own repository:

**https://github.com/ido4-dev/ido4dev**

Install via marketplace:
```
/plugin marketplace add ido4-dev/ido4-plugins
/plugin install ido4dev@ido4-plugins
```

This monorepo contains only `@ido4/core` and `@ido4/mcp` (the npm packages).
