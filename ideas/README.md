# Ideas Backlog

Parking lot for ideas, explorations, and future directions. Each idea is a separate file with frontmatter.

## Status Definitions

- **idea** — Raw concept, worth remembering. No commitment.
- **exploring** — Actively being discussed or researched. Not ready for implementation.
- **ready** — Understood well enough to become a spec or task. Waiting for prioritization.
- **parked** — Deliberately deferred. Good idea, wrong time.
- **rejected** — Explored and decided against. Kept for context on why.

## Categories

- **platform** — Core engine, architecture, infrastructure
- **integration** — External tool connections, signal providers, webhooks
- **governance** — BRE, quality gates, compliance, audit
- **experience** — Skills, agents, prompts, UX of working with ido4
